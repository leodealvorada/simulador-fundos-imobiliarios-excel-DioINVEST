# 📊 Simulador de Investimentos em Fundos Imobiliários (Excel)

## 📌 Descrição
Este projeto foi desenvolvido como parte do desafio da DIO para aplicar conceitos de **Excel** na criação de uma ferramenta prática de simulação de investimentos em **Fundos Imobiliários (FIIs)**.  
A planilha permite ao usuário realizar simulações de aportes, tempo de investimento e taxa de rendimento, oferecendo uma visão clara sobre o patrimônio acumulado e os dividendos mensais.

## 🎯 Objetivos de Aprendizagem
- Criar ferramentas de simulação de investimentos em Excel.
- Aplicar cálculos financeiros como rendimento mensal e cálculo de dividendos.
- Documentar processos técnicos de forma clara e estruturada.
- Utilizar o GitHub para compartilhamento de documentação técnica.

## ⚙️ Funcionalidades da Planilha
- Entrada de dados:
  - Valor inicial investido
  - Aportes mensais
  - Taxa de rendimento mensal (%)
  - Período de investimento (meses/anos)
- Cálculos automáticos:
  - Valor total investido
  - Patrimônio acumulado
  - Dividendos mensais estimados
- Saídas:
  - Tabela de evolução mês a mês
  - Gráficos dinâmicos de patrimônio e dividendos
  - Indicadores principais (total investido, patrimônio final, dividendos acumulados)

## 🚀 Como Usar
1. Baixe o arquivo `.xlsx` deste repositório.
2. Abra no Excel.
3. Insira os valores desejados nos campos de entrada.
4. Visualize os resultados e gráficos gerados automaticamente.
